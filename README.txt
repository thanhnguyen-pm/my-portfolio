# My Portfolio - Based on Massively by HTML5 UP

This is my personal portfolio website, built using the Massively template by HTML5 UP. The template is fully responsive, clean, and modern, making it an excellent choice for showcasing my work and projects.

## Features

- Fully responsive design
- Clean and modern layout
- HTML5 and CSS3
- Cross-browser compatibility
- Customized to showcase my portfolio

## Getting Started

1. **View the Website**  
	Open `index.html` in your browser to explore my portfolio.

2. **Explore the Code**  
	The website is built using HTML, CSS, and JavaScript. Feel free to explore and learn from the code.

3. **Credits**  
	This website is based on the Massively template by HTML5 UP, released under the Creative Commons Attribution 3.0 License.

## File Structure

- `index.html` - The main page of the website.
- `generic.html` - A secondary page for additional content.
- `assets/` - Contains CSS, JavaScript, images, and other assets.
